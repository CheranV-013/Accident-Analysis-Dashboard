# Accident Analysis Dashboard

## Overview

The Accident Analysis Dashboard is an interactive Power BI project designed to analyze road accident data and uncover trends, patterns, and key insights. The dashboard enables stakeholders to monitor accident frequency, severity, locations, and contributing factors through dynamic visualizations.

## Objectives

* Analyze accident trends over time.
* Identify accident-prone locations.
* Monitor casualty and severity statistics.
* Understand contributing factors behind accidents.
* Support data-driven decision-making for road safety improvements.

## Tools & Technologies

* **Power BI**
* **Power Query**
* **DAX (Data Analysis Expressions)**
* **Microsoft Excel / CSV Data Sources**

## Key Features

### Dashboard Highlights

* Total Accidents Overview
* Total Casualties Analysis
* Fatal, Serious, and Slight Accident Breakdown
* Monthly and Yearly Trend Analysis
* Road Type Analysis
* Urban vs Rural Accident Distribution
* Vehicle Type Impact Analysis
* Interactive Filters and Slicers

## KPIs

* Total Number of Accidents
* Total Casualties
* Fatal Casualties
* Serious Casualties
* Slight Casualties
* Accident Severity Rate
* Location-Based Accident Analysis

## Business Insights

The dashboard helps:

* Identify high-risk accident zones.
* Understand accident severity distribution.
* Track seasonal accident patterns.
* Evaluate the impact of road and vehicle categories.
* Improve road safety planning and resource allocation.
